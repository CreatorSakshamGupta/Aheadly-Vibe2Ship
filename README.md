Aheadly

Aheadly is an AI-powered productivity assistant developed for the Vibe2Ship Hackathon 2026.

The platform helps users organize tasks, prioritize work intelligently, manage procrastination, and generate emergency action plans when deadlines are approaching.

Features

* Smart task management
* AI-based task prioritization
* Personalized schedule generation
* Rescue Mode for urgent situations
* Productivity analytics dashboard
* Procrastination detection
* Offline demo mode
* Gemini AI integration

Built With

* React
* TypeScript
* Vite
* Google Gemini API
* Google AI Studio
* Chart.js
* HTML5 & CSS3

Live Application

https://aheadly-371493507306.asia-southeast1.run.app

Problem Statement

The Last-Minute Life Saver

Build an AI-powered productivity companion that proactively assists users in planning, prioritizing, and completing tasks before deadlines are missed.
